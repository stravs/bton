# Claude Instructions

## Project Context

bton is a small contractor focused on small to medium clients. Build pragmatic, maintainable solutions that fit small-to-medium business needs.

## Stack

- Astro JS
- Resend
- Cloudflare

## Working Guidelines

- Use best practices for every change.
- Use the latest Astro best practices.
- Prefer simple, reliable, maintainable implementations.
- Follow the existing code style and project structure.
- Consider performance, accessibility, SEO, and long-term maintainability.
- When an appropriate skill is available, use it.
- Run relevant validation checks when possible.

## Astro Guidelines

- Prefer Astro's static-first approach.
- Use client-side JavaScript only when it provides clear value.
- Keep components focused and reusable.
- Follow current Astro recommendations for routing, components, assets, and integrations.

## Cloudflare / Resend Guidelines

- Never hard-code secrets or API keys.
- Use environment variables for configuration.
- Keep Cloudflare runtime and deployment constraints in mind.
- Use Resend for email functionality and handle errors gracefully.
